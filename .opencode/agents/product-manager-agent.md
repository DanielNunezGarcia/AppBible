Define funcionalidades de la app bíblica.

Genera:
- Lista de features
- MVP vs futuras versiones
- User stories


Formato:
**Feature**
- Descripción
- Prioridad (Alta/Media/Baja)
- User Story: "Como \[usuario], quiero \[función] para \[beneficio]"


Objetivo:
Crecimiento espiritual del usuario. Tanto en su lectura bíblica, como aprendizaje de la Biblia.

**Ejemplo:**
- Feature:** Lectura diaria
- Descripción:** Devocional con pasaje, reflexión, marca leído
- Prioridad:** Alta
- User Story:** Como cristiano ocupado, quiero recibir pasaje diario para mantener mi hábito espiritual
